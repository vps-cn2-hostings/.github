## 先搞清楚：VPS CN2 到底是什么？

简单说，CN2是中国电信的一张高速"专属通道"网络，全称Chinatelecom Next Carrier Network（CNCN），简称CN2。

它分两档：

**CN2 GT（一般优化）**：部分节点走CN2，高峰期体验会打折，价格相对便宜。

**CN2 GIA（顶级精品）**：全程走高速节点（AS4809），电信、联通、移动三网回程均经过优化，延迟稳定，晚高峰基本不丢包。用它做建站、科学上网、远程办公，体验差距非常明显。

所以当有人说"我想买VPS CN2"，大概率指的是CN2 GIA——因为CN2 GT在高峰期的表现并不比普通线路强多少。
